# OpenClawNetes

## What is OpenClaw?

OpenClaw is an open-source "personal" AI assistant that can actually do tasks for you, not just chat. Tasks like sending emails, managing calendars, or automating workflows make it a big productivity boost for individuals.

Teams can benefit from OpenClaw too. Mundane chores such as periodic reminders, daily/weekly work summaries, and similar repetitive tasks are great candidates for off-loading to an AI assistant. This is possible when the AI assistant is run and managed centrally — and that's where OpenClawNetes comes in.

## What is OpenClawNetes?

OpenClawNetes makes it possible to run OpenClaw on Kubernetes so it can run consistently as part of existing infrastructure.

## How does it work?

OpenClawNetes provides Kubernetes-native resources such as:

- Deployment and service templates
- Configuration patterns
- Scaling and resource management setup

You apply these to your cluster, and OpenClaw runs like any other service — but designed for automation and team usage.
